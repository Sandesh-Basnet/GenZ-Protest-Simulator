# 🎮 GenZ Protest Simulation (Unity Project)

## 📌 Overview

This project is a narrative-driven simulation built in Unity, inspired by real protest events in Nepal. It presents the experience from a protester’s perspective, allowing users to explore the environment, atmosphere, and emotions of a public demonstration. The focus is on storytelling, immersion, and interactive exploration.

---

## 🎯 Purpose

This project was developed as an academic game development assignment to:

* Explore narrative design through interactive environments
* Simulate real-world scenarios in a controlled digital space
* Represent the perspective of a protester in a social context
* Gain practical experience with Unity and 3D environment design

---

## 🛠️ Built With

* Unity (3D)
* C#
* Unity Starter Assets (Third Person Controller)
* Free 3D assets (environment & props)

---

## 🎮 Features

* Single perspective gameplay (Protester viewpoint)
* Simplified Parliament-style environment
* Interactive exploration
* Basic character movement and camera control
* Atmospheric scene design

## ⬇️ How to Download & Run

### 🔹 Option 1: Clone Repository

```bash
git clone https://github.com/Sandesh-Basnet/GenZ-Protest-Simulator/
```

### 🔹 Option 2: Download ZIP

* Click the green **Code** button on GitHub
* Select **Download ZIP**
* Extract the files

---

## ▶️ How to Play

1. Open the project in **Unity Hub**
2. Load the main scene
3. Click **Play** in the Unity Editor
4. Controls:

   * **W / A / S / D** → Move
   * **Mouse** → Look around
